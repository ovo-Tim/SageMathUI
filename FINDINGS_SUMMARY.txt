  Bootstrap sizes (latest 2026.03.08-r1):
  ├─ aarch64:  30.7 MB
  ├─ arm:      27.6 MB
  ├─ i686:     29.9 MB
  └─ x86_64:   30.5 MB

  Embedding method:
  ├─ Assembly incbin (termux-bootstrap-zip.S)
  ├─ Linked as .so shared library
  ├─ Extracted via JNI at runtime
  └─ Zip entries extracted with proper permissions

  ├─ https://github.com/termux/termux-app/blob/master/app/src/main/java/com/termux/app/TermuxInstaller.java

3. BUILD SYSTEM INTELLIGENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Key Android cross-compile flags:
  ├─ ac_cv_file__dev_ptmx=yes              (Android has /dev/ptmx)
  ├─ ac_cv_func_wcsftime=no                (avoid encoding errors)
  ├─ ac_cv_func_ftime=no                   (timeb.h missing)
  ├─ ac_cv_buggy_getaddrinfo=no            (assume fixed)
  ├─ ac_cv_posix_semaphores_enabled=yes    (force posix)
  ├─ --with-system-ffi --with-system-expat (use Termux headers)
  └─ Target: clang + NDK cross-compiler triple

  Python: 13 patches applied for Android compat
  └─ Reusable patterns for other builds

  Boost: Compiled with "target-os=android" flag
  └─ Threading enabled, filesystem statx disabled

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
