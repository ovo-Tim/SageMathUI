## 🔧 Troubleshooting Reference

### If Build Fails with NDK Error
```bash
# Check NDK installation
ls ~/Library/Android/sdk/ndk/

# If empty, NDK still downloading (expected ~30-60 min)
# Check again later or try building again

# If needed, can manually install:
sdkmanager "ndk;25.2.9519653"
```

### If Build Fails with ZIP Error
```bash
# Verify bootstrap ZIP

# Should output: "All OK"
```

